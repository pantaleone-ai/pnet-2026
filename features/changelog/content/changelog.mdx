    ## Version 4
    **Released:** December 2025

    [Live Demo](https://hiretimsf.com/)

    Built with Next.js 16 (App Router), Tailwind CSS v4, and Shadcn UI. A minimal, high-performance portfolio and blog featuring a clean design, dark mode support, and a fully functional MDX blog.

    **Tech Stack:** Next.js 16, TypeScript, Tailwind CSS v4, Shadcn UI, Fumadocs, Nuqs, Jotai, PostHog

    ## Version 3
    **Released:** November 2025

    [Live Demo](https://hiretimsf.vercel.app/)

    **Tech Stack:** Next.js, TypeScript, Tailwind CSS, Shadcn UI

    ![Portfolio v3](/images/apps/web/portfolio-app-v3.jpg)

    ## Version 2
    **Released:** March 2025

    [Live Demo](https://tim-portfolio-web-v2.vercel.app/)

    Built with Next.js 15 and TailwindCSS. A modern, fast, fully responsive portfolio website with SEO optimization, MDX blog system, and custom react motion animations.

    **Tech Stack:** Next.js, TypeScript, Tailwind CSS, Shadcn UI, TanStack Query, FumaDocs, Content Collection, MDX, Nuqs, Framer Motion, Resend
