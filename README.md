# [hiretimsf.com](https://hiretimsf.com)

A minimal portfolio to showcase my work as a Frontend Developer.

Check out the live site: [hiretimsf.com](https://hiretimsf.com)

## Overview

### Stack

- Next.js 16
- Tailwind CSS v4
- shadcn/ui
- Motion
- Fumadocs

### Features

- Clean & modern design
- Light/Dark themes
- SEO optimized ([JSON-LD schema](https://json-ld.org), sitemap, robots)
- Analytics with [Vercel Analytics](https://vercel.com/analytics)

## Development

### Prerequisites

- Node.js 20+
- pnpm (recommended) or npm

### Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The site will be available at [http://localhost:1408](http://localhost:1408).

### Scripts

| Command                | Description                  |
| ---------------------- | ---------------------------- |
| `npm run dev`          | Start development server     |
| `npm run build`        | Build for production         |
| `npm run start`        | Start production server      |
| `npm run preview`      | Build and preview production |
| `npm run lint`         | Run ESLint                   |
| `npm run format:write` | Format code with Prettier    |
